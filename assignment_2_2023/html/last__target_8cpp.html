Detailed Description

Implementation of a ROS node to track the last goal target set by the user and provide it via a service.

Author
Younes Hebik
Date
03/03/2024

This node subscribes to the "reaching_goal/goal" topic to track the last target coordinates set by the user. It also provides a service to return the last target coordinates when requested.

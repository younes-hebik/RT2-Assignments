This node, named \char`\"{}\+Node\+A\char`\"{}, is responsible for interacting with an action server to set new targets, cancel goals, and monitor the robot\textquotesingle{}s position and velocity. It subscribes to feedback and odometry topics and publishes custom messages with the robot\textquotesingle{}s position and velocityyyyy.
